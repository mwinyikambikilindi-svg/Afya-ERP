# Logo-Enabled Official Report Shell Pack

This pack upgrades the official PMU/MUSE-style report shell to include:
- government logo on the left
- PMU/Police logo on the right
- centered official report heading block

## Expected static files
Place these images in your project:
- app/static/img/gov_logo.png
- app/static/img/pmu_logo.png

If your second logo is Tanzania Police Force rather than PMU, you can still use:
- app/static/img/pmu_logo.png
or rename the template path after pasting.
