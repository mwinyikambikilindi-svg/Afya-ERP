# Official Report Navigation Pack

This pack adds navigation access to the new official report routes.

## Goal
Users should stop typing URLs manually.

## What is included
- sidebar patch guidance
- control panel template patch
- optional quick-links block for dashboard/report pages

## Official on-screen routes
- /reports/official/trial-balance
- /reports/official/statement-of-financial-position
- /reports/official/statement-of-financial-performance
- /reports/official/budget-vs-actual
