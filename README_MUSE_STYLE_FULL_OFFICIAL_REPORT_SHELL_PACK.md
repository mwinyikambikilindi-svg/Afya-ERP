# MUSE-Style Full Official Report Shell Pack

This pack standardizes the core report print shell for AFYA ERP / PMU.

## Design intent
- government-style centered header
- PMU vote/sub-vote identity
- facility-aware header line for facility-specific reports
- strict financial table layout
- consistent footer
- reusable shell for multiple reports

## Recommended first targets
- Trial Balance
- Statement of Financial Position
- Statement of Financial Performance
- Budget vs Actual
- General Ledger
- Cash Receipt Register
- Cash Payment Register
