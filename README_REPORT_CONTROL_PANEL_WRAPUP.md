# Report Control Panel + Final Suite Wrap-Up Pack

This pack adds a single entry page for the PMU reporting suite.

## What it does
- creates a central report control panel
- gives quick access to:
  - Trial Balance
  - Statement of Financial Position
  - Statement of Financial Performance
  - Changes in Net Assets / Equity
  - Cash Flows
  - Budget vs Actual
  - Notes
- supports fiscal year picker
- supports direct navigation to the existing report routes

## Goal
Stop relying on memorized URLs and give the suite a more complete finished feel.
