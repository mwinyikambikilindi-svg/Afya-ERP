# PMU Note-by-Note Custom Layout Pack

This pack upgrades the note presentation layer from a generic note detail page into PMU-style note schedules.

## Included
1. PMU note layout helper service
2. Note route enhancement guide
3. Custom templates for common PMU notes:
   - Note 23: Cash and Cash Equivalents
   - Note 24: Receivables
   - Note 25: Inventories
   - Note 26: Property, Plant and Equipment
   - Note 27: Payables and Accruals
   - Note 10/12/14/16/17/18/19/20: Revenue and Expense style notes
4. Fallback generic note template

## Goal
Make notes look closer to the PMU statement style rather than a raw generic schedule.

## Important
This pack does not break your existing `/reports/notes/<note_no>` flow.
It adds a selector layer so specific notes can render with specific PMU templates.
