# PMU Official PDF Layout Hardening Pack

This pack improves the printable financial statement layouts so they look closer to a formal PMU statement.

## Included
- enhanced printable statement template
- enhanced printable trial balance template
- print CSS notes
- PMU-style heading hierarchy

## Goal
Make browser Print-to-PDF outputs cleaner and more formal while preserving the current reporting engine.

## Notes
This pack does not change calculations. It only improves presentation for print/PDF.
