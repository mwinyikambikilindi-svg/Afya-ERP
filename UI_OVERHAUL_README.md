
AFYA ERP Full UI/UX Overhaul Pack

What is inside:
- app/templates/base.html
- app/templates/*.html (major forms, registers, detail pages, students, assets)
- app/static/css/app.css
- app/static/js/app_ui.js
- dashboard.html

How to apply:
1. Backup your current app/templates and app/static first.
2. Replace app/templates/base.html.
3. Copy all templates from this pack into app/templates.
4. Copy app/static/css/app.css and app/static/js/app_ui.js.
5. Run the system again.

Notes:
- This pack is primarily a UI/UX redesign and should not change business logic.
- The CSS is intentionally compact to reduce oversized inputs, buttons, and excessive spacing.
